# Storylines

This folder holds draft narrative directions for Ambition.

The engine should remain story-agnostic: movement, collision, input, generated audio, particles, enemies, and procedural content should not depend on one canonical plot. Storylines should be layered on top through data, area rules, dialogue/events, ability unlock framing, world-state transforms, and generated scenario specs.

A storyline can define:

- the player's identity and motivation;
- factions, human collaborators, or conduit roles;
- ethical pressure systems;
- ability-unlock metaphors;
- world mirroring/topology rules;
- mathematical discoveries represented as mechanics;
- endings or long-form progression arcs.

A storyline should not require prerendered assets or bespoke engine behavior unless that behavior is promoted into a reusable Ambition Engine primitive.

## Current drafts

- [`primary_ai_agency.md`](primary_ai_agency.md): the current primary draft about an AI discovering embodiment, agency, mathematics, collaboration, and ethical compromise.

## Draft status

Everything here is provisional. The first playable priority remains the endgame movement sandbox. Story documentation exists now so narrative intent does not get lost while we iterate on feel, controls, audio, particles, and engine structure.
