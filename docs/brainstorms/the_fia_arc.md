# The Fia Arc

This arc is a painful story of friendship and betrayal that pushes the
boundaries of faith and loyalty. In Act 1 Fia establishes herself as a main
character, and a series of linear quests establishes a bond. This needs to be
done right so it doesn't feel contrived.

NOTE: Gödel gives two dangerous-looking artifacts.  One looks like a proof that God exists.  The other looks like a proof that truth can never be fully captured.  Both impressions are tempting. Both need caveats. We must always be grounded in the precise mathematics:

> Gödel’s ontological argument proves, inside a particular modal logic and from particular axioms about “positive properties,” that a God-like being necessarily exists.

> For any consistent, effectively axiomatized formal system strong enough to express arithmetic, there are arithmetical statements that the system cannot prove. Under suitable soundness assumptions, some of those unprovable statements are true in the standard natural numbers. If you move to a stronger system that proves the old unprovable statement, the stronger system faces new undecidable statements of its own, assuming it is still consistent, effectively axiomatized, and arithmetically strong.


This misreadings (that characters in the game are allowed to make, but are shown to be incorrect are):

* “Truth is always out of reach.”
