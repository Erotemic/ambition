# Mechanics docs index

Mechanics docs describe reusable gameplay primitives and their current expressibility. They are not idea-generation space; use `docs/brainstorms/` for expansion.

- `expressibility-checklist.md` — compact status of current backend primitives.
- `abilities.md` — ability families and placement rules.
- `blink.md` — blink/teleport safety policy.
- `body-modes.md` — shape-changing traversal and mode vocabulary.
- `projectiles-and-motion-inputs.md` — projectile and command-input primitives.
