## Operating rules

These govern how the canon is maintained. The blockquotes are Jon's verbatim
words; the bullets articulate them and are blessed by Jon (2026-06-03).

> For this list you can only enter things that I've written verbatim.

> This cannon is only subject to change if I say so.

> we will write a companion file that has expansions to this cannon that we can
> consider potential directions.

> I don't want to write the entire story myself.

- **Verbatim only.** Every fact/principle entry is Jon's exact words —
  attributed and dated, never paraphrased, summarized, or AI-authored.
- **Jon-gated.** This file changes only when Jon says so. The AI never adds,
  edits, or removes a canon entry on its own.
- **Propose in the companion.** Candidate directions, interpretations, and
  drafts live in `cannon_expansions.md`. Nothing becomes canon until Jon
  promotes the exact wording into this file.
- **Canon wins.** Where a story beat or expansion conflicts with canon, canon
  governs; the beat changes, not the canon.
