# Vision docs index

Vision docs describe product direction, design intent, and long-range shape.
They are active but not implementation authority by themselves; when vision
becomes a durable architecture decision, promote it to an ADR or concept page.

| Endgame sandbox | [`endgame-sandbox.md`](endgame-sandbox.md) |
| Mechanics expressibility checklist | [`mechanics-expressibility-checklist.md`](mechanics-expressibility-checklist.md) |

Use [`../brainstorms/`](../brainstorms/) for active idea development. Use this
directory for distilled direction that agents should be able to read without
scanning every brainstorm.
